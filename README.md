# GasComply AI

An AI-powered regulatory compliance and licensing platform for the Nigerian Midstream and Downstream Petroleum Regulatory Authority (NMDPRA).

## Overview

GasComply AI automates downstream gas facility licensing by combining document intelligence, machine-readable regulatory rules, engineering analysis, AI-assisted compliance review, and inspection management.

## Features

- Applicant Portal
- Engineer Portal
- Inspector Portal
- Administrator Portal
- Company Registration
- LPG Licence Management (SSA, ATC, LTO)
- AI Compliance Engine
- Engineering Intelligence Engine
- Inspection Management
- Executive Dashboard

## Technology Stack

- FastAPI
- PostgreSQL
- React / Next.js
- TypeScript
- Docker
- Kubernetes
- AI Integration
